# QR Code Tools

A simple frontend-only QR code generator built with React, TypeScript, and Vite.

## Features

- URL QR codes
- WiFi QR codes
- Email QR codes
- Phone QR codes
- SMS QR codes
- Clean multi-page layout with React Router

## Tech Stack

- React
- TypeScript
- Vite
- React Router
